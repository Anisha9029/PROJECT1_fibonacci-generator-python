# Fibonacci Generator (Python)

## Description
This is a simple Python program that generates Fibonacci numbers based on user input.  
It demonstrates basic concepts like loops, condition handling, and exception handling.

## Features
- Takes user input
- Generates Fibonacci sequence
- Handles invalid input gracefully

## How to Run
1. Clone or download the repository
2. Run the file using:
   python fibonacci.py

## Output Example
Enter how many Fibonacci numbers you want: 5  
Output: [0, 1, 1, 2, 3]
